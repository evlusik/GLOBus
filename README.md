# 📚 WPF Library Testing

## 📌 О проекте

Проект посвящён тестированию WPF-приложения «Библиотека», разработанного в рамках учебной практики.

Приложение реализует систему управления библиотекой с поддержкой различных ролей пользователей, а также функционалом работы с книгами, авторами и жанрами.

---

## 👥 Роли пользователей

### 👤 Пользователь

* Авторизация в системе
* Просмотр списка взятых книг
* Отображение количества дней до возврата
* Поиск книг по названию
* Фильтрация по жанру и году (включая комбинированную)
* Просмотр информации об авторах
* Переход к книгам выбранного автора
* Возможность взять книгу
* Редактирование профиля

---

### 📖 Библиотекарь

* Добавление книг
* Редактирование книг
* Удаление книг
* Управление авторами и жанрами

---

### 🛠 Администратор

* Управление пользователями (добавление, редактирование, удаление)
* Блокировка пользователей
* Ограничение: администратор не может удалить самого себя

---

## 🔐 Особенности бизнес-логики

* Пользователь блокируется после 3 неудачных попыток входа
* Реализована ролевая модель доступа
* Фильтрация книг поддерживает одновременное применение нескольких параметров
