# GLOBus
Данный проект посвящён тестированию WPF-приложения "Библиотека", разработанного в рамках учебной практики.

Приложение реализует систему управления библиотекой с поддержкой различных ролей пользователей и базового функционала работы с книгами, авторами и жанрами.

👥 Роли пользователей
👤 Пользователь
Авторизация в системе
Просмотр списка взятых книг
Отображение количества дней до возврата книги
Поиск и фильтрация книг (по жанру и году, включая комбинированные фильтры)
Просмотр информации об авторах
Переход к книгам выбранного автора
Возможность взять книгу
Редактирование профиля
📖 Библиотекарь
Добавление книг
Редактирование книг
Удаление книг
Управление авторами и жанрами
🛠 Администратор
Добавление пользователей
Редактирование пользователей
Удаление пользователей
Блокировка пользователей

🔐 Особенности бизнес-логики
